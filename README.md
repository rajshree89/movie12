

## Overview
This project explores a dataset of 3000 movies to guide a rookie producer in making profitable decisions. The analysis begins with a **sanity check** (missing values, duplicates, data types) and then answers six key questions.

## Key Findings
1. **Highest Profit Movie** – Identified along with its producer, director, and actors.  
2. **Language ROI** – Found the language with the highest average return on investment.  
3. **Unique Genres** – Listed all distinct genres in the dataset.  
4. **Top Producers** – Built a producer-director table and highlighted the top 3 producers by average ROI.  
5. **Most Frequent Actor** – Determined the actor with the most movies and analyzed their filmography.  
6. **Directors’ Preferences** – Revealed which actors the top 3 directors collaborated with most often.  

## Tools
- Python  
- Pandas  

## Deliverable
A structured analysis providing insights into profitable languages, genres, producers, directors, and actors to support future movie production strategies.
